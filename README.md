## 2. What is SMS-MAN

## 3. How SMS-MAN works

SMS-MAN works by assigning a temporary phone number to a user for SMS reception. SMS-MAN delivers incoming messages through a dashboard or API.

Basic SMS-MAN flow:

- SMS-MAN assigns a number

- Number expires after use

SMS-MAN also supports API-based automation, making it suitable for developers and large-scale systems.
